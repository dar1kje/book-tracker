# Веб-приложение «Учет прочитанных книг» (Book Tracker)

**Автор:** Агишев И.Н. (Группа ИНБО-12-23)
**Дисциплина:** Системная и программная инженерия (РТУ МИРЭА)

## О проекте
Минималистичный сервис для ведения личного читательского дневника, систематизации прочитанной литературы и отслеживания прогресса.

## Стек технологий
* **Frontend:** React.js, HTML5, CSS3
* **Backend:** Node.js, Express
* **Database:** PostgreSQL
